# Aplikasi Konten Otomatis

Aplikasi ini membantu membuat, mengelola, dan menyimpan konten digital secara otomatis.  
Cocok untuk membuat teks, gambar, dan konten lainnya secara cepat sesuai template.

## Fitur Utama
- Generate teks otomatis berdasarkan template
- Generate gambar sederhana (opsional)
- Menyimpan konten ke folder kategori/tanggal
- Logging konten yang dihasilkan

## Struktur Folder
- `scripts/` → Script Python / JS untuk generate konten
- `templates/` → Template teks atau gambar
- `output/` → Folder hasil konten otomatis
- `README.md` → Penjelasan project
